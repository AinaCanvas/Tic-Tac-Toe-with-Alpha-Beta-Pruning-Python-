# Tic-Tac-Toe-with-Alpha-Beta-Pruning-Python-
This project is a console-based Tic-Tac-Toe game developed in Python using the Minimax Algorithm with Alpha-Beta Pruning. The computer plays optimally against the human player by evaluating possible game states and pruning unnecessary branches to improve efficiency.
Features
Human vs Computer gameplay
AI powered by Minimax Algorithm
Alpha-Beta Pruning optimization
Input validation and error handling
Winner and draw detection
Simple console interface
Concepts Used
Artificial Intelligence
Game Trees
Minimax Algorithm
Alpha-Beta Pruning
Recursion
Python Programming
How to Run
Install Python
Download or clone the repository
Run the file:
python tic_tac_toe.py
Game Instructions
You play as X
Computer plays as O
Enter row and column values between 0 and 2

Example:

Enter row (0-2): 1
Enter column (0-2): 2
Learning Outcome

This project demonstrates how Alpha-Beta Pruning improves the performance of the Minimax Algorithm by reducing the number of nodes evaluated in the game tree.
